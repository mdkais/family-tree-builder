# Family Tree Builder

A mini family tree application built using HTML, CSS, and JavaScript.

## Features

- Add, edit, delete family members
- Parent-child relationship modeling
- Recursive family tree display
- Member detail view

## Live Demo

https://mdkais.github.io/family-tree-builder/

## Tech Stack

- HTML
- CSS
- JavaScript

## Approach

Family members are modeled as objects with references to parents and children.
The tree is rendered recursively starting from root members.
